### 批量下载得到每天听本书

#### 依赖项目
[dedao-dl](https://github.com/yann0917/dedao-dl)

#### 使用方法
1. 登录网页版获取 cookie 信息, 在 dedao-dl 中登录
2. 获取你在得到的书架中的书籍列表（需要将要下载的音频文件加到书架）
3. 将要下载的书籍信息粘贴到 down.txt 中
4. 运行 down.py 即可下载 down.txt 中的音频文件
